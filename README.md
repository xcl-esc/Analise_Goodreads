# 📚 Análise Exploratória: Dataset Goodreads

Este projeto realiza uma auditoria técnica e análise estatística de um dataset do Goodreads contendo mais de 11.000 títulos. O foco principal é a identificação de outliers, padrões de engajamento por idioma e a análise qualitativa de obras com baixa performance.

## 🚀 Objetivo
Extrair insights sobre o comportamento dos usuários e validar a integridade dos dados, estabelecendo filtros de relevância estatística para futuras modelagens.

## 🛠️ Tecnologias Utilizadas
- **Python** (Pandas, Matplotlib, Seaborn)
- **Estatística Descritiva** (Quartis, Desvio Padrão, Mediana)

## 🎯 Próximos Passos
- Análise de correlação entre número de páginas e nota média.
- Mapeamento de autores com maior consistência de qualidade.
- Análise temporal de publicações (Tendências por década).
